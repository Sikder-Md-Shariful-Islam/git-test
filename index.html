Git Installation Command

        echo "# git-test" >> README.md
        git init
        git add README.md
        git commit -m "first commit"
        git branch -M main
        git remote add origin https://github.com/Sikder-Md-Shariful-Islam/git-test.git
        git push -u origin main

Steps: 

        1. Download Git scm & Install
        2. Check Git version from Cmd
        3. Follow the command bellow: 
    
        
            1. git init

             2. git add .

             3. git commit -m "first commit"

             4. git branch -M main
             
             5. git remote add origin url

             6. git push -u origin main

To update your code in an existing repo use commands bellow:

        
            1. git add .

            2. git commit -m "second commit"
